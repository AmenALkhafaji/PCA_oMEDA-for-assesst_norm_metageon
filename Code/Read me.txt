Instructions:
	- Download MEDA package from GitHub repository through the following: https://github.com/josecamachop/MEDA-Toolbox. Please download last established version 1.6
          read the readme file to extract it in your MATLAB workspace
	- Download the Simulation & Normalization folder and place it in R 
        - run PhylumSimalution R and save the data in your path up to your drive
	- run the other codes for normalization and save the dataset in the same path the you selected before
        - You can asses the data directly ( all data with their ground truth are saved and preprepared for direct use if you want to asses phylum part please go to 
- PhylumData\Code\your method that you want to asses 
- Only Run and you are ready to go
-Or GenusData\Code\ your method that you want to asses 
- Only Run and you are ready to go
Note that files at folder CSV are the data for all methods
Nore also: all mat files in Data folder are data with the code to create them for asses where each mat has only normalized data while phylum.mat has raw data, tag,classname,groundtruth and ordered tag data
That’s all
