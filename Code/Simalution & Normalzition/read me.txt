Important Notes for Data Normalization
Before performing any normalization on your data, please ensure the following:

Load the correct raw dataset:
Ensure you are using the appropriate file at the desired taxonomic level—either Phylum or Genus.

Remove the tag column from the raw data:
The column containing sample tags or metadata should be excluded before applying normalization.

For Phylum-level data, this column is typically at position -6.

For Genus-level data, it is usually at position -37.

These steps are essential to ensure accurate and consistent normalization results.
